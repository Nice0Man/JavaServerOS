Разработка распределенной системы обработки структурированных записей

Создать процесс-сервер, который принимает запросы от клиентов и обрабатывает их, осуществляя доступ к плоскому файлу, содержащему записи одинаковой структуры. Сервер должен одновременно обрабатывать несколько клиентских запросов. Должна быть предусмотрена возможность обработки запросов: чтения, удаления, добавления и изменения.

Функциональные требования:

  • Принятие запросов от клиентов
  • Доступ к плоскому файлу с записями одинаковой структуры
  • Одновременная обработка нескольких клиентских запросов
  • Возможность выполнения операций: чтение, удаление, добавление и изменение записей

Технические детали:

  • Использование процесс-сервера для обработки запросов
  • Определение протокола для взаимодействия между клиентами и сервером
  • Реализация механизма многопоточности для одновременной обработки запросов
  • Использование базы данных или файловой системы для хранения записей
